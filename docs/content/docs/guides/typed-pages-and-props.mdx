---
title: Typed pages and props
description: Define one Todo page contract and use its generated keys in a typed test.
---

Derive `InertiaPage` when a stable page contract is shared by handlers and tests.

```rust title="src/pages/todos.rs"
use inertia_axum::prelude::*;
use serde::{Deserialize, Serialize};

#[derive(Clone, Serialize, Deserialize, InertiaType)]
pub struct Todo {
    // Use a string serializer too when IDs may exceed Number.MAX_SAFE_INTEGER.
    #[ts(type = "string")]
    pub id: u64,
    pub title: String,
}

#[derive(InertiaPage)]
#[inertia(component = "Todos/Index", rename_all = "camelCase")]
pub struct TodosPage {
    pub todos: Vec<Todo>,
    pub stats: Prop<TodoStats>,
    pub can_create: bool,
}
```

## Return the page

```rust title="src/routes/todos.rs"
use std::convert::Infallible;

async fn index() -> TodosPage {
    TodosPage {
        todos: load_todos().await,
        stats: defer(|| async { Ok::<_, Infallible>(load_stats().await) }),
        can_create: true,
    }
}
```

## Test with generated keys

```rust title="tests/todos.rs"
let page = app
    .inertia_get("/todos")
    .only(TodosPage::STATS)
    .send()
    .await
    .assert_page::<TodosPage>();

let stats: TodoStats = page.prop(TodosPage::STATS);
page.assert_missing(TodosPage::TODOS);
```

The derive generates `TodosPage::COMPONENT` and a typed key for each serialized top-level field. Use `InertiaProps` for reusable typed prop collections such as shared data; the complete attributes, rename rules, supported shapes, and reserved fields remain in the [Macros reference](/docs/reference/macros).

Run `cargo inertia sync` to compile these contracts and generate `PagePropsFor<"Todos/Index">`. See [Generated TypeScript contracts](/docs/guides/generated-types) for setup, frontend examples, integer policies, workspaces, and CI.
