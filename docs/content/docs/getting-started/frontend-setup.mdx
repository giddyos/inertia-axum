Use these commands after completing the [quick start](/docs/getting-started/quick-start).

## Development

Run Vite in the first terminal:

```bash title="Terminal 1"
pnpm --dir frontend dev -- --host 127.0.0.1
```

Point the Rust application at that server in the second terminal:

```bash title="Terminal 2"
VITE_DEV_SERVER_URL=http://127.0.0.1:5173 cargo run
```

Development mode loads assets from Vite and does not require a manifest.

## Production

```bash title="Terminal"
pnpm --dir frontend build
cargo run --release
```

Production mode reads `frontend/dist/.vite/manifest.json` at startup. The default entry is `src/main.ts`; if Vite uses another input, configure the matching `.entry(...)` on `InertiaApp::vite`.

Continue with [Vite and assets](/docs/guides/vite-and-assets) when you need custom build directories, public paths, or entries.
