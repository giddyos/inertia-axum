## Requirements

Install Rust 1.88 or newer, Node.js 22.12 or newer, and pnpm 11. The Node requirement comes from the Vite-based maintained examples.

```toml title="Cargo.toml"
[dependencies]
axum = { version = "0.8.9", features = ["http1", "tokio"] }
inertia-axum = "=1.0.0-alpha.1"
serde = { version = "1", features = ["derive"] }
tokio = { version = "1", features = ["macros", "net", "rt-multi-thread", "time"] }
```

The `time` feature supports the quick start's simulated slow query. Install the optional scaffold CLI if you want it to generate the frontend files:

```bash title="Terminal"
cargo install cargo-inertia --version 1.0.0-alpha.1 --locked
```

| Default feature | Purpose |
| --- | --- |
| `macros` | Re-exports the typed page, prop, and form derives. |
| `vite` | Reads the Vite manifest and serves built frontend assets. |

See [Cargo features](/docs/reference/cargo-features) for SSR, validation, transient storage, templates, testing integrations, and no-default-feature setups. Then build the [quick-start application](/docs/getting-started/quick-start).
