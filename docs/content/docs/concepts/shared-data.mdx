---
title: Shared data
description: Share the authenticated user with every Inertia page from an Axum request extension.
---

Use one `Share` provider for request data needed across many pages, such as the authenticated user.

## Share the current user

```rust title="src/share.rs"
use inertia_axum::prelude::*;
use std::convert::Infallible;

#[derive(InertiaProps)]
struct SharedProps {
    user: Option<CurrentUser>,
}

#[derive(Clone)]
struct AppShare;

impl Share for AppShare {
    type Props = SharedProps;
    type Error = Infallible;

    fn share(&self, context: ShareContext<'_>) -> Result<SharedProps, Infallible> {
        Ok(SharedProps {
            user: context.extension::<CurrentUser>().cloned(),
        })
    }
}
```

```rust title="src/main.rs"
let inertia = InertiaApp::vite("frontend")
    .share(AppShare)
    .build()?;
```

`ShareContext` exposes the request method, URI, headers, parsed visit, and extensions during synchronous preparation. The [authenticated shared props recipe](/docs/recipes/authenticated-shared-props) shows the complete middleware-to-client path; collision and selection rules remain in [Props and policies](/docs/reference/props-and-policies).
