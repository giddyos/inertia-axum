---
title: Page responses
description: Return the same Todo domain data through dynamic or compiler-checked page contracts.
---

Both page styles select a component and serialize props. Use `page!` for local or conditional shapes and `InertiaPage` when handlers and tests share a stable contract.

## Dynamic page

```rust title="src/routes/todos.rs"
use inertia_axum::prelude::*;

async fn todos() -> DynamicPage {
    page!("Todos/Index", {
        todos: load_todos().await,
        can_create: true,
    })
}
```

## Typed page

```rust title="src/pages/todos.rs"
use inertia_axum::prelude::*;

#[derive(InertiaPage)]
#[inertia(component = "Todos/Index", rename_all = "camelCase")]
struct TodosPage {
    todos: Vec<Todo>,
    can_create: bool,
}

async fn todos() -> TodosPage {
    TodosPage {
        todos: load_todos().await,
        can_create: true,
    }
}
```

The typed derive generates the component constant and top-level prop keys used by `inertia-axum-test`. Continue with [Typed pages and props](/docs/guides/typed-pages-and-props); conditional flags and complete macro grammar remain in the [macro reference](/docs/reference/macros).
