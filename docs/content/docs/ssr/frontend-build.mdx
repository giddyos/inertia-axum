---

Run one script to create browser assets and the Node renderer bundle from the same application entry.

```json title="frontend/package.json"
{
  "scripts": {
    "build": "vite build && vite build --ssr"
  }
}
```

## Configure Vite

<Tabs items={['Svelte', 'React', 'Vue']}>
  <Tab value="Svelte">
    ```js title="frontend/vite.config.js"
    import inertia from '@inertiajs/vite'
    import { svelte } from '@sveltejs/vite-plugin-svelte'
    import { defineConfig } from 'vite'

    export default defineConfig(({ isSsrBuild }) => ({
      plugins: [svelte({ prebundleSvelteLibraries: true }), inertia()],
      build: {
        outDir: isSsrBuild ? 'dist/ssr' : '../public/build',
        emptyOutDir: true,
        manifest: !isSsrBuild,
        rolldownOptions: isSsrBuild ? {} : { input: 'src/app.js' },
      },
    }))
    ```
  </Tab>
  <Tab value="React">
    ```js title="frontend/vite.config.js"
    import inertia from '@inertiajs/vite'
    import react from '@vitejs/plugin-react'
    import { defineConfig } from 'vite'

    export default defineConfig(({ isSsrBuild }) => ({
      plugins: [react(), inertia()],
      build: {
        outDir: isSsrBuild ? 'dist/ssr' : '../public/build',
        emptyOutDir: true,
        manifest: !isSsrBuild,
        rolldownOptions: isSsrBuild ? {} : { input: 'src/app.jsx' },
      },
    }))
    ```
  </Tab>
  <Tab value="Vue">
    ```js title="frontend/vite.config.js"
    import inertia from '@inertiajs/vite'
    import vue from '@vitejs/plugin-vue'
    import { defineConfig } from 'vite'

    export default defineConfig(({ isSsrBuild }) => ({
      plugins: [vue(), inertia()],
      build: {
        outDir: isSsrBuild ? 'dist/ssr' : '../public/build',
        emptyOutDir: true,
        manifest: !isSsrBuild,
        rolldownOptions: isSsrBuild ? {} : { input: 'src/app.js' },
      },
    }))
    ```
  </Tab>
</Tabs>

## Connect both outputs

```rust title="src/main.rs"
let inertia = InertiaApp::vite("frontend")
    .entry("src/app.js")
    .build_dir("../public/build")
    .public_path("/public/build")
    .ssr("dist/ssr/app.js")
    .start()
    .await?;
```

For this layout the client manifest is `public/build/.vite/manifest.json` and the renderer is `frontend/dist/ssr/app.js`. Production Vite requires that client manifest even though the SSR HTTP protocol does not read it, because the root document still needs browser scripts and styles and Axum serves those built assets.

Continue with [Managed Node](/docs/ssr/managed-node) or point the same frontend bundle at an [external service](/docs/ssr/external-server).
