Send the dashboard shell immediately, load summary data together, and let a nonessential activity failure recover in place.

## Server

```rust title="src/routes/dashboard.rs"
async fn dashboard() -> DynamicPage {
    page!("Dashboard", {
        greeting: "Welcome back",
        metrics: defer(load_metrics).group("summary"),
        permissions: defer(load_permissions).group("summary"),
        activity: defer(load_activity)
            .group("activity")
            .rescue(),
    })
}
```

The initial response contains `greeting`. The `summary` request resolves `metrics` and `permissions`; the independent `activity` request may be rescued without hiding the summary.

          </>}
        >
          <ul>{activity.map((item) => <li key={item}>{item}</li>)}</ul>
